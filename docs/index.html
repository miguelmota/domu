Domu.js - DOM Utilities

A collection of DOM utilities.

Documentation

http://domu.moogs.io

Install

Available via Bower

bower install domu

Extend _

To extend the _ when using libraries such as underscore or lodash, pass in domu._() to the underscore mixin function. Domu functions will not override underscore functions if they already exist, unless you pass domu._(true)
